# Embedded Systems Quiz App

This is a React-based quiz application that tests your knowledge on Embedded Systems topics. The questions are extracted from the provided PDF course materials.

## Features
- **Topic Selection**: Choose from ADC, GPIO, Digital Communication, Embedded C, and Embedded OS.
- **Multi-select Support**: Questions with multiple correct answers allow multiple selections.
- **Instant Feedback**: Review your answers and see explanations after submitting.
- **Responsive Design**: Works on desktop and mobile.

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd quiz-web-app
    ```

2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and go to: `http://localhost:5173`

## Tech Stack
- React
- Tailwind CSS v4
- Framer Motion
- Vite
